# Hospital Management Backend

## Tech Stack
- Java 17
- Spring Boot
- Maven
- Docker
- Render

## Live URL
https://hospital-backend-olti.onrender.com

## Build & Run (Local)
mvn clean package -DskipTests
java -jar target/*.jar

## Docker
docker build -t hospital-backend .
docker run -p 8080:8080 hospital-backend
